I settled on using a sidebar instead of a typical horizontal navbar so that

Navbar

I still made use of Bootstraps standard navbar component so that my sidebar navigation would be more structed


Home Screen

Card & Card-img-top

I decided to use a card that would have an image set as the background so that users
Would find the home screen of the website visually interesting upon loading. This would mean they are more likely to want to stay.

Buttons

I decided to use bootstrap buttons so that the buttons would pop and stand out to the user.

Toast and Progress

Upon clicking download i decided to add a little toast about the progress of the download
I decided to do this so that the user would have an indication from the website itself of what has happened
When they have clicked the download button

Modal

I used a modal on the contact me for the home screen even though I have a contact me section on the website.
This is because I wanted users to have quick access to contact me without it bringing them to the bottom of the website
and skipping the rest of the content

Text-Muted

I Used bootstaps text-muted class to de-emphasis some of the text on the home screen to let
Key info pop more

Icons

I made use of bootstrap icons for links to my social medias, I thought this was good
because instead of showing text I used the users familarity with these brands to get them to click the link
